# Career Sustainer

Career Sustainer is the name of this simple job-board website

## Tech Stack
- React
- TypeScript
- Vite
- HTML
- CSS
- JSON

## Features
- Posting and fetching local storage data
- Fetching APIs
- Multi-Page Package (react-router-dom)

## What I Learned
- Solutions for some bugs
- How to use APIs
- DOM manipulation
- More about react.js

## Running Locally
Open a terminal in the location of the folder you downloaded from this repository (extract if compressed) then enter:
- npm install
- npm run dev
